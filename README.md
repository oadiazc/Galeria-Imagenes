**Sobre el Proyecto**

Desarrollé esta Galería Interactiva de Imágenes con Node.js, Express, MongoDB y Bootstrap.
La idea surgió porque quería crear algo visual y funcional al mismo tiempo, un espacio donde se pudieran subir, ver y organizar imágenes de una manera sencilla y moderna.

Mi objetivo fue construir una aplicación práctica, con un diseño limpio y fácil de usar, que permita a cualquier persona subir sus propias fotos, verlas clasificadas por categorías y consultar los detalles de cada una.
Agregué categorías como Paisajes, Personas, Animales, Arte Digital y Arquitectura para mantener todo bien organizado y darle variedad al contenido.

**Características principales:**

Permite subir imágenes con título, autor, descripción y categoría.

Muestra las imágenes en tarjetas elegantes con una vista clara y atractiva.

Se pueden filtrar las fotos por categoría sin necesidad de recargar la página.

Al hacer clic sobre una imagen, se abre un modal con más información.

El diseño es responsivo, así que se adapta bien tanto en celular como en computador.
